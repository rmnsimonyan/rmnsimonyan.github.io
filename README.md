# rmnsimonyan.github.io

Personal portfolio of **Arman Simonyan** — PhD candidate at the University of Copenhagen
(Gloriam Group, Department of Drug Design & Pharmacology) working on AI-powered drug discovery
for G protein-coupled receptors.

**Live site:** https://rmnsimonyan.github.io/

## About this repository

The site is a single, self-contained `index.html` — all styling and behaviour are inlined
(CSS custom properties, Grid/Flexbox layout, vanilla-JS carousels and scroll animations).
External dependencies are loaded from CDNs only (Google Fonts, Font Awesome). There is no
build step: edit `index.html` and push to `master`, and GitHub Pages serves it directly.
